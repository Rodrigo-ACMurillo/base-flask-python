Base-flask-python

Base para proyectos desarrollados con Flask y Python. Funciona perfectamente con Heroku

Contenido

  1. Archivo de requerimientos para instalar en el servidor
  2. Archivo Procfile para ejecutar la aplicación
  3. Archivo app.py con el código de ejecución
  4. Carpeta static con los ficheros css, las imagénes y ficheros js (Contenido estático)
  5. Carpeta templates con los ficheros html de la web

Para instalar los requerimientos ingresa a la carpeta raíz y ejecuta: pip install -r requirements.txt

Para inicial la aplicación en local ejecuta en la carpeta raíz: python install, luego ingresa a localhost:5000 en tu navegador.
